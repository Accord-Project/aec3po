Description

  • Definition of operators scheme for AEC3PO.

  • A set of operators for the check method. AEC3PO includes a set of operators for this nomenclature. These operators are : - operator_exists : exists operator. - operator_notExists : not-exists operator. - operator_forall : forall operator.
