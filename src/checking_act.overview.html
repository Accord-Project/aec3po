Description

The AEC3PO Checking Act module describes the act of checking some entities for something and generating a conformance report.

Checking Act is the act of checking some entities for something and generating a conformance report. It, then, links to :

  • the check method that was used, which operationalizes a statement in a document;
  • the process verifier that made the checking act;
  • the entities that are checked;
  • Links to conformance report that resulted.
